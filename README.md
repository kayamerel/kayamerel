### 🪴 Projects

🎞️ **[ColorHead](https://colorhead.app)** - contributing to colorhead app which is a software for inverting and editing scanned color negatives

🚄 **[pyrawspeed](https://github.com/kayamerel/pyrawspeed)** - python binding to the fast RAW decoding library [rawspeed](https://github.com/darktable-org/rawspeed) which is used in ColorHead
  
🤖 **[ARM-SOFT-Grasping](https://github.com/N-Pulse/ARM-SOFT-Grasping)** - for N-Pulse association worked on computer vision pipeline for grasping using a prothesis arm

🧱 **[LegoBeans](https://github.com/epfl-cs358/2025fa-legobeans)** - for the Making Intelligent Things (CS-358(a)) course at EPFL, modified a Prusa 3D printer to function as a LEGO-compatible printer capabale of assembling BrickLink designs

👀 **[ZedToQuest](https://github.com/kayamerel/ZedToQuest) & [ZedVisionProStream](https://github.com/kayamerel/ZedVisionProStream)** - experimented with the Meta Quest 3 and Apple Vision Pro to render stereoscopic camera feed.
